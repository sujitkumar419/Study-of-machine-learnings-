# 🎓 Study of Machine Learning: Interactive Hub

An interactive, web-based Machine Learning training and revision platform designed to break down complex algorithms into simple English logic, live mathematical visualizations, and clean production-ready code.

🚀 **Developed By:** Sujit Kumar (Aspiring Data Scientist)

---

## 💡 Key Features of the Platform

- **Multi-layered Architecture:** Clean step-by-step navigation tracking from ML Types -> Specific Categories -> Individual Deep Dives.
- **Simple Conceptual Logic:** Core mathematics explained in direct, high-utility English without overwhelming technical jargon.
- **Live Parameter Simulations:** Interactive sliders allowing users to manipulate parameters (Weights, Bias, K-clusters, Contamination) and watch cost functions adjust in real-time.
- **Production-Ready Templates:** Clean, copy-pasteable `scikit-learn` and `numpy` code modules formatted like Jupyter cells for direct deployment.

---

## 🤖 Covered Machine Learning Algorithms

### 🟩 Supervised Learning
1. **Linear Regression:** Straight-line continuous prediction minimizing Mean Squared Error (MSE).
2. **Logistic Regression:** Probability-based classification utilizing the Sigmoid S-Curve.
3. **Decision Tree:** Condition-based splitting leveraging Entropy and Gini Impurity.
4. **Random Forest:** Robust Ensemble Learning combining diverse bootstrap-aggregated estimators.
5. **Support Vector Machines (SVM):** Maximum Margin boundaries utilizing flexible hyperplanes.
6. **K-Nearest Neighbors (KNN):** Real-time cluster voting based on Euclidean Distance metrics.

### 🟦 Unsupervised Learning
7. **K-Means Clustering:** Automated pattern partition using iterative centroid convergence.
8. **Hierarchical Clustering:** Granular Bottom-Up (Agglomerative) tree hierarchy using Dendrogram trees.
9. **Principal Component Analysis (PCA):** High-dimensional compression maximizing Explained Variance Ratios.
10. **Isolation Forest:** Linear time complexity anomaly detection explicitly isolating outliers.

### 🟨 Reinforcement Learning
11. **Q-Learning:** Autonomous agent loops updating Quality states using the Bellman Equation.

## 🚀 Local Installation & Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com
   cd study-of-machine-learning
   ```

2. **Install Required Libraries:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Streamlit Server:**
   ```bash
   streamlit run app.py
   ```
